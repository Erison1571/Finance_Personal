    CONTROLE FINANCEIRO - REACT APP
========================================

COMO EXECUTAR A APLICAÇÃO:

OPÇÃO 1 - Arquivo BAT Automático:
1. Clique duas vezes em "start-server.bat"
2. Aguarde o servidor inicializar (5 segundos)
3. O navegador abrirá automaticamente
4. URL: http://localhost:5173

OPÇÃO 2 - Arquivo BAT Simples:
1. Clique duas vezes em "start-simple.bat"
2. Aguarde o servidor inicializar
3. Abra o navegador manualmente em: http://localhost:5173

OPÇÃO 3 - Linha de Comando:
1. Abra o PowerShell/CMD
2. Navegue para esta pasta: cd "C:\Users\User\Documents\Projetos\Controle_Financeiro\financeiro-react"
3. Execute: npm run dev
4. Abra o navegador em: http://localhost:5173

========================================
PROBLEMAS COMUNS:

1. Porta 5173 ocupada:
   - Feche outros terminais
   - Reinicie o computador
   - Use: netstat -ano | findstr :5173

2. Dependências não instaladas:
   - Execute: npm install
   - Verifique se Node.js está instalado

3. Erro de compilação:
   - Execute: npm run build
   - Verifique os logs de erro

========================================
SUPORTE:

Se houver problemas, verifique:
1. Node.js instalado (versão 16+)
2. NPM funcionando
3. Porta 5173 livre
4. Dependências instaladas
